# 🤖 AI-Powered Conversational Interview Assistant

An intelligent AI interview simulator that conducts realistic mock interviews using conversational AI, voice interaction, and personalized feedback.

## 📌 Overview

Preparing for interviews isn't just about knowing the answers—it's about communicating them confidently. This project bridges the gap between theoretical knowledge and real interview performance by creating an AI interviewer that asks adaptive questions, understands user responses, maintains conversation context, and provides detailed feedback.

## ✨ Features

* 🎯 Multi-subject interview support

  * Self Introduction
  * Generative AI
  * Python
  * HTML
  * CSS
  * English

* 🤖 Conversational AI

  * Context-aware questioning
  * Dynamic follow-up questions
  * Natural interview flow

* 🎙 Voice-Based Interaction

  * Listen to interview questions
  * Record spoken answers
  * Hands-free interview experience

* 📝 AI Feedback System

  * Communication analysis
  * Technical evaluation
  * Confidence assessment
  * Strengths and improvement suggestions
  * Personalized recommendations

  * Browser-based

## ⚙️ Workflow

1. Select Interview Subject
2. Start Interview
3. AI asks the first question
4. User responds using voice
5. AI analyzes the response
6. AI generates a relevant follow-up question
7. Continue conversation naturally
8. End interview
9. Receive detailed AI-generated feedback
